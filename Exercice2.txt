Exercice 2 : Stocker et afficher des informations utilisateur
Ecrire l'algorithme suivant en pseudo-code :

1.Stocker un prénom
2.Stocker un âge
3.Stocker un statut de connexion
4.Afficher un message avec toutes les informations
_______________________________________________________

1.Créer variable prenom
2.Demander prenom
3.prenom <-- prenom entré

4.Créer variable nom
5.Demander nom
6.nom <-- nom entré

7.Créer variable statut_connexion
8.Demander statut_connexion
9.statut_connexion <-- statut_connexion entré

10.Créer variable message
11.message <-- prenom, nom, statut_connexion
12.Afficher variable message

13.Détruire variable prenom
14.Détruire variable nom
15.Détruire variable statut_connexion
16.Détuire variable message
